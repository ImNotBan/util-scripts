# Scripts Toolbox

Repositorio personal con scripts y pequeñas herramientas útiles.

## Objetivo
Este repositorio tiene como finalidad:
- Automatizar tareas habituales
- Aprender scripting en distintos lenguajes
- Comprender mejor el funcionamiento de sistemas Linux
- Practicar conceptos básicos de seguridad y administración

## Lenguajes
Los scripts pueden estar escritos en diferentes lenguajes, según el caso:
- Bash
- Python
- Otros lenguajes de scripting cuando sea necesario

## Estructura
Los scripts se organizan por lenguaje o por finalidad, según convenga.

## Uso
Cada script incluye instrucciones básicas de ejecución.  
Revisar siempre el contenido antes de ejecutarlo.

## Nota
Este repositorio tiene un propósito educativo.  
No ejecutar scripts con privilegios elevados sin entender su funcionamiento.
